# ratpack-thymeleaf3

Ratpack module to use Thymeleaf 3.

This is my temporal workaround until official repo supports version 3.

```groovy
repositories {
    maven {
        url  "https://dl.bintray.com/minebreaker/test"
    }
}
dependencies {
    compile 'rip.deadcode:ratpack-thymeleaf3:0.1'
}
```
